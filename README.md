# Enterprise Identity & Access Management using Microsoft Entra ID

This project demonstrates a real-world IAM implementation using Microsoft Entra ID (Free Tier).

## Features Implemented

- User lifecycle management
  
  Created and managed cloud-only users in Microsoft Entra ID following enterprise identity lifecycle practices.
  
  Created multiple users: Employee, HR user, IT Admin, Temporary Contractor, Configured user properties and Tested sign-in.
  
  <img width="1919" height="786" alt="image" src="https://github.com/user-attachments/assets/2e59969a-fb02-4d5a-9eb4-012a6ab93f5c" />

- Group-based access control
  
  Implemented group-based access control using Security and Microsoft 365 groups to simplify authorization management.
  
  Groups created: IT-Team, HR-Team, Finance-Group
  
  Actions:Assigned users to groups, Used Security & Microsoft 365 groups and Followed least privilege principle.

- Role-Based Access Control (RBAC)
  
  Assigned built-in administrative roles using RBAC to delegate user management securely.
  
  Roles assigned: User Administrator and (Other basic admin roles if any)

- Multi-Factor Authentication (MFA)-Authentication methods & strengths
  
  Enforced Multi-Factor Authentication (MFA) and configured authentication methods to enhance account security.
  
  Configured Microsoft Authenticator : Password + Microsoft Authenticator (Push Notification)
